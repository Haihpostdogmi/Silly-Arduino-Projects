# Silly Arduino Projects 

Welcome to my silly little Arduino projects! <3

Each one is my personal spin on learning a different part of Arduino — wiring, code, timing, sensors, and
more — built with the help of AI as I figure things out. They're goofy on purpose, but every one taught me something real. 

Learning by making. :D

## How this repo is organized

Each project lives in its own numbered folder, and inside you'll find:
- the Arduino sketch (`.ino`)
- a `README.md` explaining what it does and how it's wired
- a demo video or photo where available (I'm figuring this part out, so coming soon!)

## Projects so far

### 01_AkaKiroColorMixing 🎨🧡
Two RGB LEDs synced to the viral "Aka Kiro" color-mixing song from the Japanese
kids' show DokoDemoJamboree. The LEDs fade, hop, and mix colors in time with the
beat — my first dive into PWM, color mixing, and matching code timing to music.
